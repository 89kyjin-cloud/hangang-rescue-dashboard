# HanRiver Environment Dashboard v1.0 Phase 2

한강 교량별 수위·방류량·조석·조류 방향 분석을 위한 데이터 엔진 검증판입니다.

## Phase 2 반영 사항
- API 인증키 기본 숨김 처리 및 보기/숨김 버튼 추가
- 교량 목록 확장: 잠실철교, 한강철교, 당산철교, 월드컵대교, 마곡대교 추가
- 데이터 오류 방지: 최신값 fallback 제거, 기간조회 실패 시 실패로 표시
- 관측시각 차이 검증: 기본 40분 초과 시 `검증필요` 표시
- 조석 API는 사고시점~조회시점 전체 날짜 범위로 분리 조회
- 통합 변화 그래프 추가: 수위·방류량·조석을 0~100 정규화하여 비교

## 핵심 원칙
- API 실패 시 임의값으로 대체하지 않습니다.
- 수위는 수심이 아니라 관측소 기준값입니다.
- 조류 방향은 수위 변화 + 방류량 + 보정 조석 기반의 계산값입니다.
- 잠실수중보 상류 구간은 조석 적용 제외로 처리합니다.
- 공개 저장소 사용 시 API 인증키 노출에 주의하세요.
