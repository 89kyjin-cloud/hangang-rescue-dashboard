# HanRiver Environment Dashboard v1.0 Phase 2.3 FieldFix

한강 교량별 수위·방류량·조석 데이터를 검증하기 위한 분석 보조 도구입니다.

## 이번 수정

- 한강홍수통제소 수위·방류량 응답 필드 자동탐지 강화
- 1차 후보 필드가 없을 때 원자료 첫 행 전체 키를 로그에 표시
- 수위·방류량을 0으로 임의 대체하지 않음
- 투신시점/조회시점 수위, 방류량, 조석영향, 물 방향, 물살 참고판정 유지

## 교체 파일

기존 GitHub 저장소에서는 아래 2개 파일 교체를 권장합니다.

- `index.html`
- `js/app.js`

`css/style.css`는 Phase 2.2와 동일하게 사용 가능합니다.

## 주의

물 방향·물살은 유속 실측값이 아니라 수위 변화, 팔당 방류량 지연 보정, 인천 조석 보정값을 조합한 참고판정입니다. 실제 수색 지시나 작전 범위 결정은 현장 지휘와 공식 절차를 따라야 합니다.
